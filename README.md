# Aryan Dwivedi Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects, and experience as a final-year Computer Science and Engineering student at PSIT. Built with **React** and **Tailwind CSS**, the portfolio is fully responsive, visually engaging, and optimized for all devices.

## Features

- **Hero Section**: Introduction with a background image and call-to-action button.
- **About Section**: Highlights my academic background and professional interests.
- **Projects Section**: Showcases my web development and software projects.
- **Skills Section**: Highlights programming languages, frameworks, and tools I work with.
- **Experience Section**: Lists internships, programs, and other relevant experiences.
- **Contact Section**: Provides a simple way to get in touch via email or social links.
- Fully responsive and mobile-friendly design using **Tailwind CSS**.
- Smooth scrolling, modern UI interactions, and section animations.
